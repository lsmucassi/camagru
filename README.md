# camagru

> A web selfie app that allows you to stream and take pictures then edit them with png's
> post the captured pictures and allow other users to like and comment on your post
 #### functionalities
  * capture
  * edit
  * merge
  * delete
  * post
  * like
  * comment
  * create acc
  * log in and out
  * email sending
  * passwords handling
  * and more ...
